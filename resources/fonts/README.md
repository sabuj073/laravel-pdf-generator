# Bangla Font for PDF

এই ফোল্ডারে **Noto Sans Bengali** ফন্ট রাখুন। এই ফন্ট বাংলা ও ইংরেজি দুটোই সাপোর্ট করে।

## ইনস্টল করুন

```bash
php artisan pdf-generator:install-bangla-font
```

অথবা ম্যানুয়ালি:

1. https://fonts.google.com/noto/specimen/Noto+Sans+Bengali
2. Download করুন
3. ZIP এক্সট্রাক করে `NotoSansBengali-Regular.ttf` কপি করুন এই ফোল্ডারে
4. অথবা `storage/fonts/` এ রাখুন এবং `.env` এ পাথ দিন
